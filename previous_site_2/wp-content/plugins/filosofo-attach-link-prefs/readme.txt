== Description ==

Remembers your attachment "Show:" and "Link to:" preferences, so that you don't have to re-select the same ones repeatedly.

== Installation ==

1. Upload `filosofo-attach-link-prefs.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it---now your upload link preferences will be remembered by your browser.
